# 🚀 Ramya Shopping Cart (E-Commerce Web Application)

This is a **Java-based E-Commerce Web Application** customized and maintained by **Ramya**.

---

## 📌 About the Project

This project is an online shopping cart system where users can:

- Register and login
- Browse products
- Search and filter items
- Add products to cart
- Increase or decrease quantity
- Checkout products
- View order history

Admin features:

- Add new products
- Update product details
- Remove products
- Manage stock
- Track orders
- Update shipping status

---
